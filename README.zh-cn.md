个人自用的漏洞环境搭建平台，用于记录分析过的一些漏洞环境，以`docker-compose`的形式呈现，便于记录和漏洞复现。

许多经典和热门漏洞的环境都可在[vulhub](https://github.com/vulhub/vulhub)项目中找到，但难免有所缺失。个人懒惰，所以对于其没有的环境，会收录在本项目中，并尽量同时公开`Dockerfile`，以便查验和学习。
